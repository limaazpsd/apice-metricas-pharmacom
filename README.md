# 📊 Métricas Pharmacom Labs

Apresentação interativa de análise de desempenho do Instagram da **Pharmacom Labs**, desenvolvida pela **Ápice Digital**.

---

## Sobre o Projeto

Apresentação em formato de slides desenvolvida em React + Vite, com design premium inspirado em dark mode profissional.

- **6 slides interativos** com navegação por teclado (← →) e toque (swipe)
- **Canvas fixo 1920×1080** com escala CSS automática para qualquer resolução
- **Design System** compartilhado (`design.tsx`) com tokens de cor, sombra e componentes
- **Hover effects** em todos os cards com transições suaves
- **Header e Footer** fixos, sem piscar ao trocar de slide

## Slides

| # | Slide | Conteúdo |
|---|-------|----------|
| 01 | Introdução | Capa com métricas resumidas do período |
| 02 | Visão Geral | Views, Alcance, Atividade, Seguidores |
| 03 | Audiência | Gênero, Localização, Horários de Pico |
| 04 | Conteúdo | Top 3 Posts, Foco de Entrega, Atenção Crítica |
| 05 | Expansão | Importância dos Reels e Retenção via Stories |
| 06 | Conclusão | Acertos, Plano de Ação e Grande Desafio |

**Período analisado:** 28 de Maio a 06 de Julho de 2025

---

## Como Rodar

```bash
npm install
npm run dev
```

Acesse em `http://localhost:3001`

## Tecnologias

- React 18 + TypeScript
- Vite
- Tailwind CSS
- Lucide React (ícones)
- Fonte: Sora (Google Fonts)
